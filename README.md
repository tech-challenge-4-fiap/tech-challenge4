# tech-challenge4
Tech Challenge é o projeto que engloba os conhecimentos obtidos em  todas as disciplinas da fase. Esta é uma atividade que, em princípio, deve ser  desenvolvida em grupo. Importante atentar-se ao prazo de entrega, pois trata-se  de uma atividade obrigatória, uma vez que sua pontuação se refere a 90% da  nota final.
